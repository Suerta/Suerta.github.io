Linux命令之压缩命令

一、gzip[.gz]

压缩后的格式：*.gz

解压1：gunzip FileName.gz

解压2：gzip -d FileName.gz

压缩：gzip FileName

二、tar[.tar.gz/.tgz]

命令选项 参数说明
-z(gzip) 用gzip来压缩/解压缩文件
-j(bzip2) 用bzip2来压缩/解压缩文件
-v(verbose) 详细报告tar处理的文件信息
-c(create) 创建新的档案文件
-x(extract) 解压缩文件或目录
-f(file) 使用档案文件或设备，这个选项通常是必选的.

解包：tar zxvf FileName.tar

打包：tar czvf FileName.tar DirName

对于.tar.gz/.tgz

解压：tar zxvf FileName.tar.gz

压缩：tar zcvf FileName.tar.gz DirName

压缩多个文件：tar zcvf FileName.tar.gz DirName1 DirName2 DirName3 …

三、zip[.zip/.z]

与gzip相比：1）可以压缩目录； 2）可以保留原文件；

命令选项 参数说明
-r(recursive) 递归压缩目录内的所有文件和目录

解压：unzip FileName.zip

压缩：zip -r FileName.zip DirName

Z命令：

解压：uncompress FileName.Z

压缩：compress FileName

.tar.z命令：

解压：tar zxvf FileName.tar.z

压缩：tar zcvf FileName.tar.z DirName

四、bzip2[.bz2/.bz/.tar.bz]

压缩后的格式：.bz2

命令选项 参数说明
-k 产生压缩文件后保留原文件

解压1：bzip2 -d FileName.bz2

解压2：bunzip2 FileName.bz2

压缩： bzip2 -z FileName

.tar.bz2命令

解压：tar jxvf FileName.tar.bz2

压缩：tar jcvf FileName.tar.bz2 DirName

bz命令

解压1：bzip2 -d FileName.bz

解压2：bunzip2 FileName.bz

压缩：未知

.tar.bz命令

解压：tar jxvf FileName.tar.bz

总结：

对于压缩后缀中有tar的，解压和压缩命令都是(z/j)xvf或 (z/j)cvf

压缩：tar zxvf FileName.tar.* 或 tar jxvf FileName.tar.*

解压：tar zcvf FileName.tar.* 或 tar jcvf FileName.tar.*

对于压缩后缀中没有tar的，但是是zip的不同类型，解压命令都是＋un/-d，除了zip，如下：

zip: unzip FileName.zip 或者 zip -r FileName.zip

gzip: gunzip FileName.gz 或者 gzip -d FileName.gz

bzip: bunzip FileName.bz 或者 bzip -d FileName.bz
